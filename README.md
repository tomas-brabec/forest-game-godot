# forest-game-godot
- 2D plošinovka kde se hráč snaží dostat domů přes různé překážky
- Hra je vytvořena pomocí herního enginu [Godot](https://godotengine.org/)
- Projekt vznikl v rámci sedmidenního game jamu a jeho hratelná verze je dostupná [zde](https://linein98.itch.io/forest)
